\doxysubsection{Detailed Description}
Implementation of the \doxylink{class_polygon}{Polygon} class used to represent 2D polygons. 

This file contains the definitions of the member functions of the \doxylink{class_polygon}{Polygon} class, which encapsulates a 2-\/dimensional polygon defined by an array of \doxylink{class_point}{Point} instances representing its vertices.
